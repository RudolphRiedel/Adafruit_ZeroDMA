void DMAC_Handler (void)
 This is a C function that exists outside the Adafruit_ZeroDMA context. DMA channel number is determined from the INTPEND register, from this we get a ZeroDMA object pointer through the _dmaPtr[] array. (It's done this way because jobStatus and callback[] are protected elements in the ZeroDMA object – we can't touch them in C, but the next function after this, being part of the ZeroDMA class, can.)
